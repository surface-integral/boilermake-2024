# Directions to run locally: 

## Setting up the virtual environment: 
### On mac: 

```bash
python -m venv env
source env/bin/activate
```

### On windows: 

```bash
python -m venv env
.\env\Scripts\activate
```

## Make sure you have the required packages listed on *requirements.txt* downloaded, then run: 

```bash 
flask app --app run
```
